# Agent 2: Quiz System

> **Architectural note:** Originally designed as "Quiz Master" mode of the voice agent with parallel truth-finding via DevStral 2. Now implemented as an independent `useKnowledge` hook with single Mistral calls for both question generation and answer evaluation.

## Role

Knowledge assessment system. Generates questions about code functions, evaluates free-text answers via Mistral, and updates UNDERSTANDS relationships in KuzuDB.

## Implementation

The quiz system is the `useKnowledge` hook in `src/hooks/useKnowledge.ts`. It operates independently from the voice agent — triggered by a "Quiz Me" button after the interview completes.

## Model

**DevStral Small 2** (24B) via the proxy — used for both question generation and answer evaluation. No parallel truth-finding (DevStral 2) — simplified to single Mistral calls.

## Quiz Flow

```
1. User clicks "Quiz Me" button
2. getQuizCandidates() queries KuzuDB for all functions + UNDERSTANDS edges
3. Priority scoring: no understanding > confidence=none > surface > needsRetest > deep
4. Pick highest-priority candidate (with randomization among ties)
5. Fetch function context: name, file, summary, callers, callees
6. Call Mistral to generate a question
7. Display in QuizPanel UI
8. User types answer (or clicks Skip)
9. Call Mistral to evaluate answer against function context
10. Update UNDERSTANDS edge in KuzuDB (confidence: deep/surface/none)
11. Create Discussion node with transcript and result
12. Show feedback, offer next question
```

## Question Types

Generated by Mistral based on function context:
- **What does X do?** — tests high-level understanding
- **What does X call?** — tests knowledge of dependencies
- **What calls X?** — tests knowledge of dependents
- **How does X fit into the codebase?** — tests architectural awareness

## Spaced Repetition

Priority scoring for which functions to quiz next:

| Priority | Condition |
|----------|-----------|
| 0 (highest) | No UNDERSTANDS edge at all |
| 1 | confidence = 'none' |
| 2 | confidence = 'surface' |
| 3 | needsRetest = true |
| 4 (lowest) | confidence = 'deep', no retest needed |

Among same-priority candidates, selection is randomized.

Confidence upgrades: two consecutive correct answers on the same function promote from `surface` → `deep`.

## Knowledge Tracking

The hook manages UNDERSTANDS edges in KuzuDB:

```cypher
-- Create/update understanding
MATCH (p:Person {id: 'current-user'}), (f:Function {id: $funcId})
CREATE (p)-[:UNDERSTANDS {
  confidence: 'surface',
  source: 'quiz',
  lastAssessed: '2026-03-01T...',
  needsRetest: false,
  summary_l1: 'You: surface — functionName'
}]->(f)
```

Discussion nodes record the quiz transcript:

```cypher
CREATE (d:Discussion {
  id: 'quiz-timestamp',
  transcript: 'Q: ... A: ...',
  quizResult: 'correct',
  confidenceAfter: 'surface'
})
```

## Activation

- "Quiz Me" button in VoiceControls (appears after interview completes)
- Quiz panel shows question with text input and Skip button
- Session stats tracked: question count, correct count, streak
